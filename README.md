# Εφαρμογή Κατανομής Μαθητών Αʼ Δημοτικού
Αυτή η εφαρμογή βασίζεται σε Streamlit και υποστηρίζει:
- Ποίημα και κοινωνικό μήνυμα
- Κατανομή βάσει παιδαγωγικών κριτηρίων (εκπαιδευτικοί, φίλοι, ιδιαιτερότητες, φύλο)
- Γραφήματα και στατιστικά
- Εξαγωγή Excel με λογότυπο

## Εκτέλεση:
```bash
streamlit run app.py
```
